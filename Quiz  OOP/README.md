Data yang harus diisi adalah:
1. E-mail (harus sesuai format, misal: agungnhd@upnyk.ac.id)
2. Username (hanya boleh menggunakan huruf dan angka)
3. Password (wajib menggunakan huruf, angka dan simbol)
